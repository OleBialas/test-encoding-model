## Installation

Clone the repo and run `pixi install`:
```
git clone https://github.com/OleBialas/test-encoding-model.git
cd test-encoding-model
pixi install
```

## Data

Download the file [mSM63_09-Aug-2018.mat](https://drive.google.com/drive/folders/1tLr6DpzlrCjIA95QeYPNDErx1eZh5ZA-) and put it into the `data/` folder.
